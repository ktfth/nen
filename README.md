# nen

Skills para Claude Code focadas em segurança de aplicações (AppSec).

## Instalacao

```bash
npx skills add ktfth/nen
```

## Skills Disponiveis

### AppSec Orchestrator

Pacote completo de skills para trabalho de AppSec defensivo ponta-a-ponta.

| Fase | Skills |
|------|--------|
| Intake | Intake, Scope Guardrails |
| Threat Model | Threat Model Lite, Attack Surface Map |
| Review | Code Review, Dependency Review, Secrets Review |
| Testing | SAST Baseline, DAST Safety, AuthZ/AuthN Checks, Injection Checks, Business Logic |
| Fixes | Fix Playbook, Secure Defaults |
| Validation | Regression Plan, Security Gates |
| Delivery | Report, Risk Acceptance, Handoff |

## Uso

Apos instalar, as skills estarao disponiveis no Claude Code. Consulte a documentacao em `skills/appsec/README.md` para detalhes sobre cada skill e modos de execucao.
